v2.2.6 (July 2026)

This version brings numerous improvements and enhancements to the Forumslader Garmin data field project, focusing on user experience, feature set, and configuration flexibility. Key changes include expanded display/recording field options, improved default settings, and new features such as automatic trip reset.

Feature and Settings Enhancements:

- Additional display/recording fields: consumption, range, BLE diagnostics
- New "Auto Trip Reset" option that automatically resets the Forumslader's daily trip counter (km) when an activity is started on the Garmin
- Choice between a large single-field display, switchable by tapping, and a compact multi-field display
- Colored live icon for battery capacity and charging state, always visible
- Forumslader's daily and tour distance counters (km) can be reset via the Garmin IQ menu
- Forumslader parameters wheel circumference, dynamo pole count, and auto-off delay are configurable
